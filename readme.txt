Пример стандартного проекта под Visual Code. 

Подробнее параметры для сборки и настройки ищите в папке .vscode в файлах launch.json и task.json.

Для корректной работы в среде Visual Code необходимо:
1. Установить gcc с с99 и прописать путь к нему в PATH
2. Установить gdb (обычно в комплекте c gcc и по тому же пути) и прописать пути к нему в launch.json
3. Установить расширение C/С++ от Microsoft
